Principais formatacoes

Negrito

Nao semantico usando a tag <b>;

texto em destaque usando <strong>

Italico

usando nao sematico usa se a tag <i>

usando semanticamento usa se a tag <em>

Texto marcado

para um texto marcada usa se a tag <mark>

agora outra tag marque

Texto grande e texto pequeno

voce pode escrever um texto grande usando a tag <big> ou um texto pequeno usando a tag <small>

Texto deletado

para texto deletado usa se a tag <del>

Sublinhado

usando a forma semantica usa se a tag <ins>

usando a forma nao semantica usa se a tag <u>

texto elevado ou texto abaixo

para o texto elevado usa se a tag <sup> um exemplo eh 210 = 1024

para o texto abaixo usa se a tag <sub> um exemplo disso eh a molecula da agua H2O
